# Harmony Home Furnishings — Version 8

A modern, AR‑powered furniture storefront built with HTML, CSS, and JavaScript.  
Includes AR previews using `<model-viewer>`, a dynamic cart system, search, filters, and category pages.

---

## 🚀 Features

- Home page with hero + category grid  
- Living Room, Bedroom, Dining, Outdoor category pages  
- AR‑ready products using `.glb` + `.usdz`  
- Dynamic product rendering from `products.js`  
- Cart with quantity, remove, and localStorage persistence  
- Search bar + search results page  
- Price + rating filter bar  
- Product detail page  
- Fully responsive modern UI
